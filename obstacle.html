Obstacle Robot

Autonomous Navigation & Collision Avoidance

Project Overview

This autonomous vehicle was built by my students under my coordination, designed to navigate complex environments without human intervention.

The core of its intelligence lies in its Ultrasonic Sensor system. It emits sound waves that bounce off obstacles, allowing the robot to calculate distances in real-time using mathematical algorithms. When an object is detected within 10 cm, the robot instantly halts and autonomously recalculates a new path to avoid collision.

Robot Specs

  • Vision: Ultrasonic Echolocation
  • Detection Range: < 10 cm
  • Logic: Real-time Path Correction
  • Drive: Differential Steering
  • Role: Student Project Coordinator
